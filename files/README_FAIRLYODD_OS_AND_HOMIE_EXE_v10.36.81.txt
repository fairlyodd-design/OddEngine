FairlyOdd launcher files

- RUN_FAIRLYODD_OS_AND_HOMIE_v10.36.81.bat
  One-click launcher that starts the OS services and desktop UI.

- BUILD_FAIRLYODD_OS_AND_HOMIE_EXE_v10.36.81.ps1
  Uses Windows built-in IExpress to generate:
  FairlyOdd_OS_and_Homie_v10.36.81.exe

How to build the EXE on Windows:
1. Open PowerShell in C:\OddEngine
2. Run:
   powershell -ExecutionPolicy Bypass -File .\BUILD_FAIRLYODD_OS_AND_HOMIE_EXE_v10.36.81.ps1
