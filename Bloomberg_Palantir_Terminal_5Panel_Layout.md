
# OddEngine Terminal Layout (Bloomberg / Palantir Style)

To instantly feel like a high‑end terminal, use a 5‑zone layout.

## Layout Grid

Top Bar
Market tape / alerts

Left Rail
Navigation + workspace switcher

Center Main Panel
Primary workspace (Trading, Grow, etc.)

Right Intelligence Rail
AI assistant, alerts, analytics

Bottom Activity Stream
System events and logs
