Homie Neural Voice Bridge v10.36.86

This is the honest "start designing an AI clone of myself" pass.

What it does now:
- local HTTP bridge on 127.0.0.1:8776
- clone-profile JSON file you can edit
- preview endpoint that shapes text into a more "you"-like tone
- emotion blend and cadence presets
- gesture suggestion (wave / wink / nod / tilt / spark)
- request receipts for debugging

What it does NOT pretend to do:
- it does not magically create a neural clone voice by itself
- for real neural audio, set HOMIE_NEURAL_TTS_ENDPOINT to a local or trusted provider you control
