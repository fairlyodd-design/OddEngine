# v10.25.46 — Phoenix Replay + Validation Pass

This contained SAFE_OVERLAY adds a real replay-and-validation lane to the Coinstore BTC/USDT Futures Phoenix panel.

## What it adds
- Replay scrubber with step back, step forward, and play/pause controls
- Replay frame reconstruction from the active chart using scaled slices across the full timeframe stack
- Outcome grading for each replayed setup: TP1, TP2, STOP, OPEN, or WAIT
- Homie grading on each replay: right, early, late, or wrong
- Validation scoreboard with win rate by setup type and recent replay outcomes
- Quick-jump buttons to the best long and best short setup buckets
