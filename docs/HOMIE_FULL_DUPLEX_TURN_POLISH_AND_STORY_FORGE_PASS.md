## Homie Companion

This pass tightens the continuous voice-chat handoff so the open mic feels smoother and less browser-limited.

- mic-reopen smoothing state between turns
- speech-finished watchdog for browsers that fail to emit a clean synth end event
- improved loop recovery copy and state badges

## Writers / Studio Pipeline

This pass upgrades the Books panel into a prompt-to-finale creative pipeline.

### Added
- Story Forge prompt builder
- book / movie / mini movie / cartoon / music video / song project typing
- full room-by-room packet generation using Studio Home, Writing Room, Director Room, Music Lab, Render Lab, and Producer Ops
- final packet export as markdown
- per-room asset preview and export
- persistent local project metadata for prompt, style, release target, budget band, scope, and finale checklist

## Honest note

The Writers upgrade gives you a strong local-first production packet and project-management flow. It does not pretend to generate the fully rendered movie/song/book by itself yet; it prepares the work cleanly for the next human or render-system step.
