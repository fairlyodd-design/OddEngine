# v10.38.21_HomieTrue3DAvatarAssetAndViewerPass

## Mission

Move Homie from PNG collage mode into a starter true 3D avatar lane.

## Phase 1 included here

- Adds `HomieTrue3DAvatar.tsx`.
- Canvas-rendered procedural 3D-style avatar:
  - hoodie body
  - jeans
  - shoes
  - cap
  - beard
  - glasses
  - glowing core
- Drag to rotate.
- Idle turn.
- Subtle breathing.
- Mood hook prop.
- PNG/CSS stage remains underneath as fallback.
- Main Homie + HomieBuddy receive the same 3D stage where anchors are found.

## What this is not yet

This is not a final rigged GLB/VRM. It is a Phase 1 procedural 3D viewer lane that creates the interaction model now and gives us a clean place to swap a real GLB/VRM later.
