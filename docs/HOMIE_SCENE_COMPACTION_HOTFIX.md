# v10.26.4b Homie Scene Compaction Hotfix

This hotfix tightens the Homie Companion window so the fallback buddy fills more of the stage and the window stops wasting so much vertical space.

## What changed
- moved the camera closer and aimed it lower
- scaled the fallback buddy up so Homie reads as a full companion instead of a tiny figure
- raised the stage ring so the body sits better in frame
- reduced the scene panel height
- compacted the lower status card so it does not repeat the full presence copy

## Goal
Make the default Homie Companion feel more like one cohesive buddy window and less like a tall debug scaffold.
