- makes Homie bigger

1. Close OddEngine and Homie Companion.
2. Copy this hotfix over your current `OddEngine` folder.
3. Start `RUN_HOMIE_PAIR_DEV_WINDOWS.bat` again.
4. Homie should look bigger and the window should feel tighter.
