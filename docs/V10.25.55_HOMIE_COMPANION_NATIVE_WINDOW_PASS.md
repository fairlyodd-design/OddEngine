# v10.25.55 — Homie Companion Native Window Pass

This pass upgrades the Homie companion from a renderer popup into a real Electron/native BrowserWindow in desktop mode.

What it adds:
- native detachable companion window
- true outside-the-main-window movement
- always-on-top toggle routed to Electron
- saved native companion size/position
- second-monitor placement preference
- fallback popup remains for web/no-shell mode

Apply steps:
1. copy overlay into repo root
2. run `node .\scripts\apply-homie-companion-native-window.mjs`
3. run `npm --prefix .\ui run build`
4. restart desktop mode
