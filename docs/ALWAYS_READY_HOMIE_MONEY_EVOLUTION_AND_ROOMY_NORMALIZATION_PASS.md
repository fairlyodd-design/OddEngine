# v10.26.12n — Always-Ready Homie + Money/Homie Evolution + Roomy Normalization

## What changed
- Homie Buddy now opens in always-ready mode by default and exposes a typed chat lane directly inside the floating companion.
- Added a new Preferences toggle for keeping Homie always open with the animated avatar.
- Embedded Homie Core now uses an animated companion orb treatment, quick voice buttons, and a Money/Homie pulse card.
- Money now includes a Homie income huddle card that can hand the best move or a step-breakdown prompt into the Homie panel.
- Added broader panel-by-panel roomy normalization rules for Money, Brain, Homie, DevEngine, Preferences, Security, Family Budget, and Mining.

- FairlyOdd OS needs Homie to feel like a real companion, not a helper you have to hunt for.
- Money mode works better when Homie can coach the next move straight from the shell and the Money board.
- Several pages still collapsed too late into cramped card grids, so this pass makes them breathe earlier.
