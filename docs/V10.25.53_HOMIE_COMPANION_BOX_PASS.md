# v10.25.53 — Homie Companion Box Pass

## What this pass adds
- detachable Homie companion window
- in-app floating companion box
- saved size and position
- idle / alert / coach / listening states
- voice/listening button with local speech shortcuts
- toast handoff from Phoenix alerts
- optional second-monitor popup mode
- bundled default Homie image asset

## Honest note
This is a SAFE_OVERLAY. The popup companion is a real detach window, but OS-level always-on-top still needs a shell/electron main-process hook if you want it to stay above every other application.
