# v10.37.8d_GlobalShellAsciiIconSanityPass

## Mission

Clean mojibake that spread beyond FairlyGodMode into the global shell.

## Fixes

- Sanitizes `ui/src/App.tsx` visible shell text to ASCII-safe output.
- Replaces `PANEL_META` icon values in `ui/src/lib/brain.ts` with ASCII-safe short labels.
- Adds CSS so ASCII labels fit inside nav icon boxes.
- Normalizes the pin button to a plain `*`.
- Saves patched files as UTF-8 without BOM.

## Why

Emoji and Unicode symbols were being corrupted into mojibake such as `a...`, `A...`, and `dY...` through the Windows/PowerShell patch path. This pass removes risky glyphs from the global shell until a safer icon system is added later.

## Safety

No shell layout rewrite.
No Trading logic changes.
No CardGODMode logic changes.
No Homie backend, voice, or memory changes.
No panel behavior changes.
