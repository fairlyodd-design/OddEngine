# v10.26.12h — Calendar Full Width Board Pass

## Goal
Make the Calendar panel feel like a true command board by giving the month grid the full available panel width.

## What changed
- Converted Calendar to a full-width single-column layout.
- Moved the selected day, upcoming, and linked panels cards below the month board.
- Increased calendar cell minimum width and height so day cells breathe more.
- Kept the existing event editing and panel deep-linking behavior intact.

## Result
The month board now gets center-stage width, with support cards living underneath instead of stealing horizontal space from the main calendar.
