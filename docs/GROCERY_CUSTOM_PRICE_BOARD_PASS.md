# v10.26.6b Grocery Custom Price Board Pass

This pass adds a custom-list price board to Grocery Meals so a fresh grocery list can be pasted in without replacing the usual planner list.

Included:
- custom list textarea
- per-item cheapest lane display
- coupon lane display
- one-store estimate display
- open-best-store and open-coupon-lane actions
- add-priced-list-to-planner action
- local save/restore of the custom list draft
