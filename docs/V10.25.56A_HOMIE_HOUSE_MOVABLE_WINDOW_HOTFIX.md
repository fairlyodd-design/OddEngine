# v10.25.56a — Homie House Movable Window Hotfix

This hotfix fixes two usability problems in detached mode:
1. The detached Homie House was difficult or impossible to move because it opened frameless and transparent.
2. It stayed on top too aggressively because always-on-top was effectively the default.

The detached window now opens with a native frame and normal background, and always-on-top defaults off.
