# v10.38.18_HomieGeneratedPortraitAssetSwapPass

## Mission

Switch Homie from a pure CSS-built face to a repo-safe generated PNG portrait asset for a stronger visual jump.

## What this adds

- `ui/public/homie/homie-reference-portrait.png`
- `ui/public/homie/homie-reference-portrait-256.png`
- CSS overlay that makes the generated portrait the lead visual.
- CSS avatar remains underneath as fallback.
- Main Homie and HomieBuddy companion use the same portrait direction.

## Visual target

- Closer to the provided reference image.
- Smooth white cap.
- Glasses.
- Warm eyes.
- Open smile.
- Pointed beard.
- Soft 3D character portrait feel.
- Still clearly Homie / FairlyOdd.

## Safety

- No backend rewrite.
- No voice engine internals.
- No Trading changes.
- No CardGODMode changes.
- No shell changes.
- No copied external assets or branding.

## Apply

```powershell
cd C:\OddEngine
.\RUN_APPLY_HOMIE_GENERATED_PORTRAIT_ASSET_SWAP.bat
npm --prefix ui run typecheck
npm --prefix ui run build
npm run dev:desktop
```

Then tag:

```powershell
git add .
```
