# Homie Embodied Companion Pass

This pass keeps the restored main look and feel intact while making Homie feel more like a living AI companion in the mascot spot.

## Goals

- keep the existing face/look direction
- make Homie feel more present and responsive
- add visible mode, readiness, and recent-room context
- keep Home / Homie / Preferences aligned with companion-first behavior

## Visible changes

- Homie mascot now shows mode + presence state
- bubble includes companion lead/body, readiness chips, and recent-room memory
- Home check-in lane reads more like a companion presence card
- Homie deck language shifts from helper-first to embodied-companion-first
- Preferences labels move away from old Lil Homie wording
