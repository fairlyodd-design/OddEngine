# v10.26.6a Grocery Deal Action Polish Pass

This pass tightens Grocery Meals so the list feels more like a real savings cockpit instead of a rough planner scaffold.

## What changed

- Quick add now splits comma-separated text into real items.
- Manual entry labels now read as quick-add instead of manual add.
- Each grocery item surfaces a best current lane badge.
- The item action board now supports cheapest lane, coupon lane, and one-store lane picks.
- Open at store follows the chosen lane for the item.
- Best basket summary is slightly tighter and more decision-focused.

## Goal

Help the operator answer three questions fast:

1. What should I buy?
2. Where should I buy it?
3. What is the cheapest clean move right now?
