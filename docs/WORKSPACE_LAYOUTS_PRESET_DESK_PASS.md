# Workspace Layouts Preset Desk Pass

Adds true saved workstation layouts on top of detachable panel windows.

## What changed
- adds built-in desk presets: Trading Desk, Family Ops, Studio, and Money Hunt
- adds a custom workspace builder in the left rail
- saves named multi-panel layouts locally
- reopening a layout relaunches its saved panel set
- each panel still remembers its own last window bounds for trader-style freeform placement
- desktop mode can tile a workspace using the existing routine window engine
- desktop mode can close the active workspace and reset remembered window bounds
- main workspace bar now exposes quick preset launch buttons

## Operator flow
1. Pop panels out and park them where you want.
2. Save the panel set as a named workspace.
3. Reopen that workspace later and each panel comes back in its remembered place.
4. Use Tile when you want an instant structured desk layout.
