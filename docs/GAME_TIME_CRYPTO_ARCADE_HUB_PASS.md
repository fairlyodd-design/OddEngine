# v10.26.11v Game Time Crypto Arcade Hub Pass

## What this pass does

- upgrades the existing Games / ZBD panel into a broader **Game Time** hub
- keeps the workflow **manual-play only**
- adds an official-source layer for:
  - ZBD Earn
  - THNDR Games
  - WOMPLAY
  - sMiles
  - Fumb Games
- adds a starter catalog import so the user can get a broad crypto-game board quickly
- keeps local sats/session tracking and emulator helpers
- preserves prior saved state by loading legacy `oddengine:cryptoGames:v2` data and saving to both `v3` and legacy `v2`

## Main user-visible changes

- panel title becomes **Game Time**
- mounted sections:
  - Scout Board
  - Official Sources
  - Library
  - Sessions & Sats
- reward/source filters in Library
- starter import buttons per official source hub
- broader seeded catalog under one roof instead of only a tiny ZBD-only seed

## Important truth

This pass is a legit discovery / launch / tracking workflow. It does **not** automate gameplay or attempt to farm third-party rewards.
