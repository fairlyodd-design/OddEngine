# v10.26.11 Homie Companion Conversation Pass

Adds a real companion chat lane inside `homie_companion` so Homie can respond like a warm co-pilot.

## Included
- Companion chat thread
- Quick replies
- Local keyword-guided supportive responses
- State changes tied to conversation (`listening`, `talking`, `alert`)

## Notes
- This is a local companion conversation layer, not cloud AI.
- It is meant to make Homie feel alive and helpful while the deeper voice/live-intelligence lanes keep growing.
