# v10.26.4 Homie Polish and Presence Pass

This pass smooths the separate Homie Companion app instead of adding a second window or more risky runtime branches.

## What changed

- better default Homie window size and launch position
- drag-friendly header bar for easier moving
- safer first-show behavior so the window appears after the renderer is ready
- presence strip and friendlier status language
- scene loading overlay instead of awkward dark transitions
- smoother fallback and loaded-avatar motion blending
- tighter control dock labels
- small Grocery Meals polish bonus in the OddEngine UI

## Grocery bonus

The Grocery Meals tab now includes:

- trip mode cards: Quick refill, Family stock-up, Dinner saver
- a simpler house mission note
- a "Tonight's move" card so the grocery lane feels more guided

## Best run path

1. Start Homie Companion.
2. Start OddEngine.
3. Probe the bridge in Preferences.
4. Keep one Homie window only.
