# v10.26.4a Homie Full Body Framing Hotfix

This hotfix widens the Homie Companion camera framing so the fallback buddy reads as a true full-body presence instead of a cropped half-body peek.

## Changes
- Moved the default camera back and widened FOV
- Added a stable camera look target
- Lowered the stage ring/floor so the feet have breathing room
- Nudged fallback and loaded avatar root positions upward for better portrait framing

## Goal
When Homie launches in the default floating window, the whole buddy should fit in view without the body getting chopped off.
