# v10.26.4a_HomieFullBodyFramingHotfix — like I'm 5

## What changed
Homie was standing too close to the camera, so only the top half showed up.

We fixed that by:
1. moving the camera back
2. making the camera see a little wider
3. lowering the floor ring
4. nudging Homie into a better standing spot

## How to use it
1. Close Homie Companion and OddEngine
2. Copy this overlay on top of your current OddEngine folder
3. Start `RUN_HOMIE_PAIR_DEV_WINDOWS.bat` again
4. Homie should show much more of the body in the window
