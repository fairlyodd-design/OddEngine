# Grocery Retail Runner And Coupon Clip Pass

Version: 10.26.7c

This pass adds a stronger store-run instruction layer to Grocery Meals.

## What changed
- Added a runner + coupon clip guide for custom grocery trips
- Per-item active lanes now show clearer "clip here", "buy here", or "wait here" actions
- Added store-by-store runner cards with clip/buy/wait counts
- Added queue actions for runner-ready items

## Goal
Turn the local deal scout into a more practical shopping coach so the user can see which lanes to clip first, which items to buy now, and which ones to hold for a better coupon window.
