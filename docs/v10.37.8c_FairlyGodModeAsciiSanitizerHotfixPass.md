# v10.37.8c_FairlyGodModeAsciiSanitizerHotfixPass

## Mission

Repair the FairlyGodMode mojibake/emoji issue with an ASCII-only PowerShell script.

- Script file is ASCII-only.
- Strips non-ASCII characters from `ui/src/components/FairlyGodModeHUD.tsx`.
- Replaces workspace mode icons with safe labels:
  - Legacy
  - Trade
  - Money
  - Studio
  - AM
  - PM
  - House
- Removes emoji rendering from FairlyGodMode deck panel title spots.
- Saves the file as UTF-8 without BOM.

## Safety

No shell changes.
No Trading changes.
No CardGODMode changes.
No Homie backend, voice, or memory changes.
No behavior change besides text/icon cleanup.
