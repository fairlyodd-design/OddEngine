v10.36.11_SingleScrollShellAndStaticRailsPass

Purpose
- make the center lane the single true scrolling surface
- keep left/right rails visually planted
- freeze right-side buddy/rail motion that can create breathing or compositor wobble

Use
1. unzip over C:\OddEngine
2. run RUN_v10.36.11_SingleScrollShellAndStaticRailsPass.bat
3. restart OddEngine

Cleanup
- after the patch is applied, you can delete the downloaded zip
- you can also delete the RUN_, PATCH_, and README files from C:\OddEngine if you want
- the styles.css changes remain in place after those helper files are removed
