# v10.26.14n Lil Homie Reference Personalization Pipeline Pass

1. Copy this pass over your current build.
2. Drop your real face / hoodie / cap images into `ui/public/models/references/`.
3. Copy `ui/public/models/lilhomie.hero.identity.template.json` to `ui/public/models/lilhomie.hero.identity.json`.
4. Edit the colors and style notes.
5. Run:
   `python scripts/lilhomie_reference_pack_builder.py --front ui/public/models/references/front-face.png`
6. Run:
   `python scripts/build_lilhomie_glb.py`
7. Launch desktop mode and open detached Homie.
8. For the full custom Blender rebuild, run `scripts/lilhomie_blender_personalization_helper.py` inside Blender.
