v10.26.19b_UniversalFreeformPanelDockingAndDetachableWorkspacePass

What changed:
- Upgraded FairlyGOD card layout toolbar into a more visible Customize bar.
- Added quick actions for all panels:
  - Float all
  - Dock all
  - Spread
  - Roomy
  - Freeform
  - Pop panel
- Added app-wide panel popout event bridge so Customize -> Pop panel opens the active panel in its own window.
- Improved floating card polish so widgets behave more like detachable desk tools.
- Relaxed panel canvas clipping and added more breathing room for freeform layouts.
- Updated version.ts to 10.26.19b.

How to use:
1. Open any panel.
2. Use the Customize bar at the top-right of the panel.
3. Hit Freeform to break cards out into movable floating widgets.
4. Hit Spread to auto-lay cards out across the workspace.
5. Hit Pop panel to open the current panel in its own desktop/browser window.
6. Use Dock all to put everything back into normal flow.
