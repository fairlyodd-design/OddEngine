# v10.26.19a One Prompt To Final Artifact Studio Pass

## What changed
- Writers Lounge now opens with **Magic Studio** first.
- You can type one prompt like:
  - write a 300 page novel
  - make a short cartoon
  - create a movie pack ready for audience release
- The OS now:
  - detects what you want
  - builds a chapter / scene / part plan
  - shows the full production path
  - gives a final artifact pack target
  - lets Homie help finish it
  - can bridge the active run into the creative backend

## What to click
1. Open **Books / Writers Lounge**.
2. Stay on **Magic Studio**.
3. Paste one creation prompt.
4. Click **Create magic run**.
5. Click **Autopilot active run** to complete the internal lane.
6. Click **Bridge active run** if your creative backend is live.
7. Click **Handoff to Homie** if you want polishing help.
