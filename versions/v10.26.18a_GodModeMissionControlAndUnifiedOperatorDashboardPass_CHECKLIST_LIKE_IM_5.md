# v10.26.18a God Mode Mission Control + Unified Operator Dashboard

1. Start OddEngine like normal.
2. Look for the new **God Mode** quick layout button in the top workspace bar.
3. In the God Mode shell, click **Float Operator Dashboard**.
4. You should now see one dashboard that combines:
   - Money
   - Studio
   - Trading
   - Homie
   - Mission Control
5. Use the tabs:
   - Today
   - Now
   - Queued
   - Blocked
6. Click any card to jump into the panel that needs attention.
7. If your creative backend or analytics are offline, the dashboard should show that as a blocker instead of crashing.
