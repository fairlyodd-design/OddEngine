# v10.26.12a Global Panel Grid Normalization Pass — checklist like I'm 5

1. Open your OddEngine folder.
2. Drop these updated files into the matching folders.
3. Start OddEngine.
4. Open a few different panels like:
   - Home
   - Trading
   - Family Budget
   - Books / Studio
   - Crypto Games
   - Calendar
5. In the left rail, find **Look + fit**.
6. Try:
   - **Auto-grid On / Off**
   - **Width −**
   - **Width +**
   - **Default**
7. Open **Preferences** → **Desktop Mode**.
8. You should now see new saved controls for:
   - Auto-grid
   - Grid target cell width
9. Open any panel and click the **Layout** bar.
10. Try:
   - **Auto-grid: On / Off**
   - **Width −**
   - **Width +**
   - **Width Auto**
11. Resize the panel cells and switch between a few panels.
12. You should notice:
   - less cramped summary cells
   - wider sections getting more room
   - more consistent card rhythm across the OS

If a specific panel still feels too stacked, this pass sets up the global system so the next move can be a more targeted panel-by-panel cleanup instead of another giant rewrite.
