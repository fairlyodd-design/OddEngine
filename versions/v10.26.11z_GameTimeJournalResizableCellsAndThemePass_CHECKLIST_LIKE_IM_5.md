# v10.26.11z Game Time Journal + Resizable Cells + Theme Pass — checklist like I'm 5

1. Open your OddEngine folder.
2. Drop these updated files into the matching folders.
3. Start OddEngine.
4. Open **Crypto Games**.
5. Open **Sessions & Sats**.
6. You should now see:
   - a quick capture form
   - a merged recent journal feed
   - a seven-day game rollup
   - daily pace cards
7. Try this flow:
   - pick a game
   - enter minutes
   - enter sats
   - click **Log payout** or **Log finished session**
8. Start a real session, then use **End active + save** from the journal lane.
9. Look at the left shell rail.
10. You should now see **Look + fit**.
11. Change:
   - theme
   - surface density
   - cell size
12. Open **Preferences** → **Desktop Mode**.
13. You should now see saved controls for:
   - OS theme
   - surface density
   - cell scale
   - card min height
   - rail widths
14. Open any panel and use the **Layout** bar.
15. Try:
   - **Cells −**
   - **Cells +**
   - **Cells 1.00×**
16. Drag a panel card from the lower-right corner and confirm it resizes.

If something feels too big or too tight, use the rail card or Preferences panel to dial it back instead of editing CSS by hand.
